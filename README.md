shabinary
====

It is a python package to hash binary data.

## Sample

    import shabinary
   
    seed = 1234567890
    input = shabinary.logisticmap_binary( 10, 256, seed )
   
    sb = shabinary.shabinary('sha256')
    output = = sb.binary2d( input )
